# Final Data Analytics Project - E-Commerce Business Analysis

## Project Objective
The objective of this project was to perform complete business data analysis on an E-Commerce dataset using Python, SQL, and Power BI to generate business insights and visual dashboards.

---

# Tools & Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- SQL
- SQLite
- Power BI
- GitHub

---

# Project Workflow

## 1. Data Cleaning
- Removed missing values
- Checked duplicate records
- Cleaned inconsistent data
- Prepared dataset for analysis

## 2. Exploratory Data Analysis (EDA)
- Order analysis
- Revenue analysis
- Payment analysis
- Customer behavior analysis
- Visual trend analysis

## 3. SQL Data Analysis
- SELECT queries
- COUNT analysis
- GROUP BY operations
- Revenue calculations
- Payment method analysis
- Order status analysis

## 4. Dashboard Development
Created interactive Power BI dashboard including:
- Total Orders KPI
- Total Revenue KPI
- Average Payment KPI
- Payment Method Pie Chart
- Order Status Bar Chart

---

# Key Business Insights
- Most orders were successfully delivered
- Credit cards were the most used payment method
- Revenue exceeded 16 million
- High-value transactions were identified
- Customer installment trends were analyzed

---

# Conclusion
This project improved practical knowledge of data analytics, SQL analysis, dashboard development, business intelligence, and data visualization using real-world e-commerce datasets.
