# Project Summary

This project focused on analyzing an E-Commerce dataset using Python, SQL, and Power BI.

The project included:
- Data cleaning and preprocessing
- Exploratory data analysis
- SQL-based business analysis
- Dashboard creation using Power BI

Main analyses performed:
- Order analysis
- Revenue analysis
- Payment method analysis
- Customer behavior analysis

Power BI dashboard visuals:
- Total Orders KPI
- Total Revenue KPI
- Average Payment KPI
- Payment Method Pie Chart
- Order Status Bar Chart

Technologies used:
Python | SQL | SQLite | Power BI | GitHub

This project helped improve practical skills in data analytics, visualization, and business intelligence.
