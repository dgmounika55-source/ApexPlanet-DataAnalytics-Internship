# Final Data Analytics Project

## Overview
This project is a complete end-to-end E-Commerce Data Analytics project performed as part of the ApexPlanet Data Analytics Internship.

The project combines:
- Data Cleaning
- Exploratory Data Analysis
- SQL Analysis
- Dashboard Visualization
- Business Insights

---

# Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- SQL
- SQLite
- Power BI
- GitHub

---

# Project Components

## Data Cleaning
Performed preprocessing and handled missing and duplicate values.

## Exploratory Data Analysis
Analyzed revenue, payment methods, and order trends.

## SQL Analysis
Performed business analysis using SQL queries including:
- COUNT
- GROUP BY
- ORDER BY
- SUM
- AVG
- WHERE

## Power BI Dashboard
Created interactive dashboard with:
- Total Orders KPI
- Total Revenue KPI
- Average Payment KPI
- Payment Method Pie Chart
- Order Status Bar Chart

---

# Key Insights
- Most orders were successfully delivered
- Credit cards generated highest revenue
- Revenue exceeded 16 million
- Customer payment trends were identified

---

# Conclusion
This project improved practical skills in business analytics, SQL querying, dashboard creation, and data visualization using real-world E-Commerce data.
