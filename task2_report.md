# Task 2 - Exploratory Data Analysis (EDA)

## Objective
The objective of this task was to analyze the Brazilian E-Commerce dataset and discover business insights using visualizations and statistical analysis.

## Analysis Performed

1. Payment statistics analysis
2. Payment distribution visualization
3. Payment method analysis
4. Monthly order trend analysis
5. Order status distribution analysis
6. Top 10 highest payment analysis

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code

## Key Insights

- Credit cards were the most used payment method
- Most orders were successfully delivered
- Some customers made very high-value purchases
- Order trends changed over different months
