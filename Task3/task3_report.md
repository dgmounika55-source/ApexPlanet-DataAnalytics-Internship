# Task 3 - SQL for Data Analysis

## Objective
The objective of this task was to perform SQL-based data analysis on the Brazilian E-Commerce dataset using SQLite.

## Tools Used
- SQLite
- DB Browser for SQLite
- SQL
- GitHub

## Tables Used
1. orders
2. payments

## SQL Operations Performed
- SELECT
- COUNT
- GROUP BY
- ORDER BY
- WHERE
- SUM
- AVG
- LIMIT

## Business Analysis Performed
- Total orders analysis
- Order status analysis
- Payment method analysis
- Revenue analysis
- Average payment analysis
- Top payment analysis
- Delivered and canceled order analysis
- Installment payment analysis

## Key Insights
- Most orders were successfully delivered
- Credit cards were the most used payment method
- Credit cards generated the highest revenue
- High-value payments were identified
- Customer installment patterns were analyzed

## Conclusion
This task improved understanding of SQL queries, database management, filtering, grouping, sorting, and business data analysis using SQLite.
