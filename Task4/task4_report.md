# Task 4 - Power BI Dashboard

## Objective
The objective of this task was to create an interactive business dashboard using Power BI for analyzing e-commerce sales and payment data.

## Tools Used
- Power BI
- Power Query
- Data Visualization
- GitHub

## Dataset Used
- orders dataset
- payments dataset

## Dashboard Visuals Created
- Total Orders KPI Card
- Total Revenue KPI Card
- Average Payment KPI Card
- Payment Method Pie Chart
- Order Status Bar Chart

## Business Analysis Performed
- Order analysis
- Revenue analysis
- Payment method analysis
- Customer payment behavior analysis

## Key Insights
- Most orders were successfully delivered
- Credit cards were the most used payment method
- Total revenue exceeded 16 million
- Average customer payment value was identified

## Conclusion
This task improved understanding of dashboard development, KPI visualization, business analysis, and interactive reporting using Power BI.
