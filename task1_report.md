# Task 1 - Data Immersion & Wrangling

## Objective
The objective of this task was to clean and prepare the Brazilian E-Commerce dataset for analysis.

## Steps Performed

1. Loaded multiple datasets using Pandas
2. Checked dataset structure and column information
3. Identified missing values
4. Checked duplicate records
5. Filled missing categorical values with "Unknown"
6. Filled missing numerical values using median
7. Converted date columns into datetime format
8. Saved cleaned datasets

## Tools Used
- Python
- Pandas
- VS Code

## Output Files
- cleaned_products.csv
- cleaned_orders.csv

## Key Learnings
- Data cleaning
- Missing value handling
- Data transformation
- Dataset profiling
